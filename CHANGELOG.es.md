# Registro de Cambios

Todos los cambios notables en este proyecto serán documentados en este archivo.

## [0.0.2] - 2026-06-30

### Añadido
- Modo de Consola Interactiva con almacenamiento en caché de memoria (`--interactive` o `-i`) para consultas rápidas.
- Calculadora de crianza con buscador de rutas (pathfinder) para calcular árboles de crianza multigeneracionales a partir de los Pals del jugador.
- Opciones de filtrado avanzado (rasgos pasivos, género, nivel mínimo) en el analizador de Palbox.
- Capacidades de exportación de datos a formatos CSV y JSON.
- Limpieza in-situ de contenedores de semillas y basura con copias de seguridad automatizadas.

## [0.0.1] - 2026-06-30

### Añadido
- Telemetría en tiempo real, monitoreo, crianza y búsqueda de inventario para Palworld.
- Escaneo de caché de sesión de Steam para configuración sin fricciones.
- Integración de servidor Model Context Protocol (MCP).
- Demonio de host persistente mediante el programador de tareas de Windows.
