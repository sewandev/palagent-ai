# Changelog

All notable changes to this project will be documented in this file.

## [0.0.2] - 2026-06-30

### Added
- Memory-cached Interactive Console mode (`--interactive` or `-i`) to query save files fast.
- Pathfinder Breeding Calculator to compute multi-generational breeding trees from owned Pals.
- Advanced filtering options (passive traits, gender, min-level) for Palbox analyzer.
- Data export capabilities to CSV and JSON formats.
- In-place item cleaning with automated backups.

## [0.0.1] - 2026-06-30

### Added
- Real-time telemetry, monitoring, breeding and inventory search for Palworld.
- Steam session cache scanning for zero-touch configuration.
- Model Context Protocol (MCP) server integration.
- Windows Task Scheduler persistent host daemon.
